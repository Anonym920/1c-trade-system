# 1С Торговля - Система учета торговли

## Описание

Приложение для автоматизации учета торговли включает:
- Регистрация поступлений товара на склад по накладным
- Регистрация продаж товара со склада с выпиской счет-фактур
- Учет остатков товаров по складам
- Аналитика и отчеты

## Требования

- Python 3.8+
- MySQL 8.0+
- PyQt6 (для графического интерфейса)

## Установка

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Создание базы данных

```bash
mysql -u root -p < database/schema.sql
mysql -u root -p trade_system < database/sample_data.sql
```

### 3. Конфигурация

Создайте файл `.env` в корневой директории:

```env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=trade_system
DB_PORT=3306
```

## Запуск приложения

```bash
python app/main.py
```

## Основные возможности

### 1. Управление справочниками
- Поставщики
- Покупатели
- Товары
- Склады

### 2. Управление документами
- Накладные (поступления)
- Счет-фактуры (продажи)
- Детали товаров в документах

### 3. Учет остатков
- Текущие остатки по складам
- Зарезервированные товары
- Доступное количество

### 4. Аналитика и отчеты
- Остатки товаров
- Анализ поступлений
- Анализ продаж
- Рентабельность по товарам
- Финансовые отчеты

## Использованные технологии

- **Backend**: Python 3.8+
- **СУБД**: MySQL 8.0
- **GUI**: PyQt6
- **ORM**: SQLAlchemy
- **Отчеты**: ReportLab
